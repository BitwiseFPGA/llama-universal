# llama-makcu universal
C++ recoil base for using a Makcu. An ImGui GUI is included in the project. This can be used in a 1pc / 2pc setup.

# Features
- Enable / Disable recoil
- X & Y control
- Custom delay
- Custom control time (If set to 3 seconds, will only control recoil for 3 seconds)
- Create custom recoil tables | Save and load custom recoil tables
